# 在线购物系统前端

## 技术栈

- Vue 3
- Vue Router 4
- Pinia
- Axios
- Element Plus
- Vite

## 项目结构

```
frontend/
├── src/
│   ├── api/              # API接口
│   ├── components/       # 通用组件
│   ├── router/           # 路由配置
│   ├── stores/           # Pinia状态管理
│   ├── utils/            # 工具函数
│   ├── views/            # 页面组件
│   ├── App.vue           # 根组件
│   └── main.js           # 入口文件
├── index.html
├── package.json
├── vite.config.js
└── README.md
```

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 预览

```bash
npm run preview
```
